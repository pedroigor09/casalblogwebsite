# Pasta de Fotos

## Instruções

Adicione uma foto do casal com o nome `couple-photo.jpg` nesta pasta.

### Requisitos da imagem:
- **Nome do arquivo:** `couple-photo.jpg`
- **Formato:** JPG ou JPEG
- **Resolução recomendada:** Mínimo 1920x1080px
- **Proporção:** Qualquer (será aplicado blur e coberto toda a área)

A imagem será usada como fundo blur na seção "Missão: O Grande Dia" com os anéis 3D.
